# Installation Guide

## Clone
- ### GitHub Clone
	Paste command into terminal
	> gh repo clone IttyBittyAC/AT2_Web
- ### Git Clone
	Paste command into terminal  
	> git clone https://github.com/IttyBittyAC/AT2_Web.git
- ### Visual Studio
	Open Visual Studio 2022  
    Click Clone a repository   
	In Repository location Paste 
	> https://github.com/IttyBittyAC/AT2_Web.git
