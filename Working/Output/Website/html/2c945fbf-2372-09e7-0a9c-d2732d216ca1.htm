      XMLConfigService Methods

      Methods

      GetFaq Find a faq by id (case-insensitive). Returns null if not found.
      GetFaqs Return all parsed FAQs.
      GetFeature Find a feature by id (case-insensitive). Returns null if not found.
      GetFeatures Return the cached list of features. Caller should treat the returned list as readonly.
      GetFooter Return a footer by id. The current lookup attempts to match the provided id to the first column title (case-insensitive). If no match is found the first footer is returned. Adjust lookup logic if you prefer an explicit footer id attribute.
      GetFooters Return the parsed footer sections. Currently this typically contains a single Footer.
      GetMenu Find a menu by id (case-insensitive). Returns null if not found.
      GetMenus Return all parsed menus.
